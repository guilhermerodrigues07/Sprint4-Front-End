# Sprint4

Projeto modernizado em **React + Vite + TailwindCSS**.

Principais recursos:
- TailwindCSS configurado
- CSS Grid e layout responsivo (mobile-first)
- Componentes: Header, Footer, Dashboard, Modal, Dropdown, Contact Form
- Dashboard com gráficos simples (SVG) e dados simulados
- Dark mode e paleta inspirada na imagem enviada (fundo escuro + ícones dourados)
- Deploy configurado para Vercel (`vercel.json`)

## Como rodar
```bash
npm install
npm run dev
```

## Observações
- Este repositório foi criado a partir do projeto enviado e modernizado. O histórico Git foi removido.
- Para deploy na Vercel, basta fazer link com a sua conta e utilizar os comandos padrão.
